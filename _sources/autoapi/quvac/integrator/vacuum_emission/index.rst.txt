quvac.integrator.vacuum_emission
================================

.. py:module:: quvac.integrator.vacuum_emission

.. autoapi-nested-parse::

   Calculation of vacuum emission integral (box diagram, F^4).

   Currently supports:

   1. Calculation of total vacuum emission signal for given field configuration.
   All fields are treated as external.

   2. Separation of fields into pump and probe with subsequent calculation of
   probe channel signal.

   .. note::
       For details on implementation check out :ref:`implementation` section.

.. py:function:: determine_integration_scheme(Nt, integration_method)

   Determine integration weights based on the integration scheme.

   :param Nt: Number of time points in a discretized time integral.
   :type Nt: int
   :param integration_method: Quadrature rule to use.
   :type integration_method: 'trapezoid' or 'simpson'

   :returns: Integration weights for a given time interval.
   :rtype: np.ndarray of length Nt


.. py:class:: VacuumEmission(field, grid, fft_executor=None, nthreads=None, channels=False)

   Calculator of Vacuum Emission amplitude from given fields

   :param field: External fields.
   :type field: quvac.Field
   :param grid: Spatial and spectral grid.
   :type grid: quvac.grid.GridXYZ
   :param fft_executor: Executor that performs FFTs.
   :type fft_executor: quvac.pyfftw_executor.FFTExecutor, optional
   :param nthreads: Number of threads to use for calculations. If not provided,
                    defaults to the number of CPU cores.
   :type nthreads: int, optional
   :param channels: Whether to calculate a particular channel in vacuum emission amplitude.
                    Default is False.
   :type channels: bool, optional
