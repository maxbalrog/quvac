.. py:function:: create_job_executor(cluster_type, executor_folder, sbatch_params, max_parallel_jobs)

   Create submitit job executor.

   :param cluster_type: On which machine the calculation would be performed.
   :type cluster_type: 'local' or 'slurm'
   :param executor_folder: Path where to create a folder for executor files and logs.
   :type executor_folder: str
   :param sbatch_params: Parameters of individual Slurm job.
   :type sbatch_params: dict
   :param max_parallel_jobs: Maximal number of jobs to run concurrently.
   :type max_parallel_jobs: int

   :returns: Executor for running jobs.
   :rtype: submitit.AutoExecutor
