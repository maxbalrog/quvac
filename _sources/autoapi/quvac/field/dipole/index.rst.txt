quvac.field.dipole
==================

.. py:module:: quvac.field.dipole

.. autoapi-nested-parse::

   Analytic expression for dipole wave and function to create multibeam
   configuration by compining several focused fields.

   .. [2] S. S. Bulanov et al. "Multiple Colliding Electromagnetic Pulses:
       A Way to Lower the Threshold of e+ e-Pair Production from Vacuum."
       PRL 104.22 (2010): 220404.

.. py:class:: DipoleAnalytic(field_params, grid)

   Bases: :py:obj:`quvac.field.abc.ExplicitField`


   Analytic expression for dipole wave.

   :param field_params:
                        Dictionary containing the field parameters.
                            - focus_x : tuple of float
                                Location of spatial focus (x, y, z).
                            - focus_t : float
                                Location of temporal focus.
                            - theta : float
                                Spherical angle of dipole moment d0 with z-axis (in degrees).
                            - phi : float
                                Spherical angle of dipole moment d0 with x-axis (in degrees).
                            - lam : float
                                Pulse wavelength.
                            - tau : float
                                Duration.
                            - W : float, optional
                                Energy.
                            - envelope : str
                                Envelope type, either "plane" or "gauss".
                            - dipole_type : str
                                Type of dipole, either "electric" or "magnetic".
   :type field_params: dict
   :param grid: Spatial and spectral grid.
   :type grid: quvac.grid.GridXYZ

   .. rubric:: Notes

   Dipole wave expression is from [1]_.

   d0 is along ez by default.

.. py:function:: create_multibeam(params, n_beams=6, mode='belt', phi0=0, idx0=1)

   Create multibeam configuration from several focused pulses to
   approximate the dipole wave and achieve higher intensity
   at focus.

   :param params: Dictionary containing the field parameters.
   :type params: dict
   :param n_beams: Number of beams, by default 6.
   :type n_beams: int, optional
   :param mode: Configuration mode, either 'belt' or 'sphere', by default 'belt'.
   :type mode: str, optional
   :param phi0: Initial angle for the beams, by default 0.
   :type phi0: float, optional
   :param idx0: Offset index for `beams` dictionary.
   :type idx0: int, optional

   :returns: **beams** -- Dictionary containing the parameters for each beam.
   :rtype: dict

   .. rubric:: Notes

   Configuration follows from [2]_.
