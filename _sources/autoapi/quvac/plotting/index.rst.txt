.. py:function:: plot_fields(field, t, plot_keys=None, cmap='coolwarm', cnorm='log', norm_lim=1e-20, save_path=None)

   Plot specified field components.

   Plotted figures:
   1. (x,y) and (x,z) profiles at the focus.
   2. x, y, z slices through the focus.

   :param field: Field object containing the field data.
   :type field: quvac.field.Field
   :param t: Time at which to plot the field.
   :type t: float
   :param plot_keys: List of field components to plot, by default None.
   :type plot_keys: list of str, optional
   :param cmap: Colormap, by default 'coolwarm'.
   :type cmap: str, optional
   :param cnorm: Normalization type, by default 'log'.
   :type cnorm: str, optional
   :param norm_lim: Normalization limit, by default 1e-20.
   :type norm_lim: float, optional
   :param save_path: Path to save the plots, by default None.
   :type save_path: str, optional

   :returns: **field_comps** -- Dictionary containing the field components.
   :rtype: dict
