quvac.pyfftw_executor¶

Pyfftw executor that is used in all 3D FFTs.

At the moment, batched FFT transforms of three 3D arrays are performed (corresponding to a vector field on 3D grid).

class quvac.pyfftw_executor.FFTExecutor(tmp_shape, nthreads=None, fft_axes=(1, 2, 3))[source]¶

Class to unite the FFT calculation.

Parameters:

  • tmp_shape (tuple or list) – Size of the buffer array over which FFTs are performed. By default, it expects (3, Nx, Ny, Nz).

  • nthreads (int, optional) – Number of threads to use for calculations. If not provided, defaults to the number of CPU cores.

  • fft_axes (tuple or list) – Axes over which to perform FFT. By default, (1,2,3).
